# AI识别规则

## 分类判断

| 分类 | 关键词 |
|------|--------|
| **餐饮** | 早餐、午餐、晚餐、买菜、零食、水果、奶茶、咖啡、面包、包子、饺子、面条、米饭 |
| **日常** | 洗衣液、纸巾、水电、房租、衣服、药品、交通、音乐、会员、剪头发 |
| **项目** | AI、DeepSeek、服务器、课程、软件 |
| **资产** | 买房、房款、首付、定金、购车、买车、车辆、投资、基金、股票（大额长期资产） |
| **其他** | 不属于以上 |

## 类型判断

| 关键词 | 类型 |
|--------|------|
| 买、花、支出 | `'#type/支出'` |
| 收、工资、收入 | `'#type/收入'` |
| 转账、还款 | `'#type/转账'` |

## 退款/退货特殊处理

**关键词**：退款、退货、退钱、返还

**处理方式**：仍然记为 `'#type/支出'`，金额为**正数**（表示冲减支出）

**原因**：退款是支出冲销，不是收入

## 处理流程示例

用户说"买10块钱包子"：

1. 关键词"包子" → **餐饮**
2. "买" → **支出**
3. 金额10元 → `amount: -10`
4. 生成描述"包子"
5. 当前时间 → `datetime`
6. 4个带单引号的Tag
7. 创建文件
